- refreshAll uses useRef pattern for stable callback (spoolsRef.current to avoid dependency on spools):
  ```
  const spoolsRef = useRef(spools);
  useEffect(() => { spoolsRef.current = spools; }, [spools]);
  const refreshAll = useCallback(async () => {
    const tags = spoolsRef.current.map(s => s.tag_spool);
    if (tags.length === 0) return;
    const fresh = await batchGetStatus(tags);
    dispatch({ type: 'SET_SPOOLS', spools: fresh });
  }, []);
  ```
- refreshSingle: useCallback fetches getSpoolStatus(tag) then dispatches UPDATE_SPOOL
- Export: SpoolListProvider, useSpoolList (throws if context undefined)
